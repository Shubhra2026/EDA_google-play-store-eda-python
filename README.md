# Google Play Store App Analysis

## 🎯 Objective
This project performs an exploratory data analysis on the Google Play Store dataset to understand app market dynamics, including user ratings, installation patterns, and category popularity.

## 🛠 Tools Used
* **Python**
* **Libraries:** Pandas, Matplotlib, Seaborn
* **Techniques:** Data Cleaning, Correlation Analysis, Exploratory Data Analysis (EDA)

## 📈 Key Insights
* **Correlation:** Analyzed the relationship between the number of installations and user ratings to determine if popularity influences quality perception.
* **Market Trends:** Identified dominant categories and overall user satisfaction trends within the Google Play Store ecosystem.

## 🚀 Conclusion
This analysis provides a data-driven overview of the app market, helping to visualize how apps perform based on user engagement metrics.
